Feel Food

FEEL FOOD

CONTEXT

I decided to learn by myself how to create websites and applications. For this I decided to code a web application to search for ecological food products. So I created Feel Food, an application that allows you to search for products, scan products to find out more about them and also compare products between them. This project allowed me to learn HTML, CSS, Javascript but also to manipulate data via API.

PROCESS

I made this project with the open source API open food fact. I also made all the graphics by hand using Illustrator.
